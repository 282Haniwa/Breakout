ブロック崩しのゲームです。

—————————————————————————————————————————————————————
＊キー操作
・全体
　qキー：　ゲーム終了

・スタート画面
　スペースキー：　ゲーム開始

・ゲーム画面
　スペースキー：　ボールの発射

・ゲームオーバーorゲームクリア後
 スペースキー：　スタート画面へ
—————————————————————————————————————————————————————
*仕様
ブロックを全て消すとゲームクリア
ボールを落としてしまうとゲームオーバー
ボールが急に早くなったり縦にしか動かなくなったりするのは仕様です。
